# pong-game
Classic Pong game made in Python

## Features
* Play against computer
* Hardcore mode: points scored in a row
* Save high score
* 3 levels of difficulty

## Controls
#### Singleplayer
* **Service** spacebar
* Up & down arrows

#### Multiplayer
* **Service** spacebar
* **Player 1** Z & S
* **Player 2** Up & down arrows
